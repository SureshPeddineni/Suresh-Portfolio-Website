Student-Management-System

Project Description

This is a simple and interactive Student Management System built using HTML, CSS, and JavaScript. The project allows users to add, view, edit, search, and delete student records in a structured table format. Each student record includes details such as Name, Email, Age, GPA, and Degree.

The system dynamically updates the student list in real-time without requiring any page reload. It also includes a search feature that allows users to quickly filter students by Name, Email, or Degree, and an edit mode that lets users update existing student details easily.

The user interface is clean and responsive, designed with CSS and simple layout styling.

Key Features

  • Add new student records
  • Display student details in a table format
  • Update/Edit existing student information
  • Delete student records
  • Data stored in Local Storage for persistence
  • Responsive and user-friendly interface
