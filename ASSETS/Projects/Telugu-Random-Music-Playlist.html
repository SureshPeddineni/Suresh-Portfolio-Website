Project Description

A Random Playlist App generates a unique mix of songs each time for a fresh listening experience.

It shuffles tracks from different genres to keep music sessions exciting.

Users can enjoy endless playlists with variety and surprise every time they play.

Key Features

  • Plays songs in random order automatically
  • Play, Pause, Next, and Previous controls
  • Shows song name, thumbnail/cover image, and duration
  • Smooth UI with responsive design
  • Stores song details in an array for easy customization
